Autotagging Project Poster

In this project, as part of my role, I developed a machine learning model for their market search engine. I created a classification model which can classify all the startups automatically into their trend categories by using Python. Here are the techniques and tools that I used; MySQL to collect data from database, Scrapy (CSS and XPath selectors) to scrape the textual data from websites, NLP techniques (TF-IDF, NLTK, Word2Vec) to work textual data, Logistic Regression, K- Fold cross-validation, Random Sampling techniques (to deal with imbalanced data) for classification. My model increased the baseline accuracy about %40.
